# Linaark Architects Portfolio

A stunning, modern portfolio website for an architecture firm featuring premium animations, light/dark theme toggle, and responsive design.

![Status](https://img.shields.io/badge/Status-Live-brightgreen) ![HTML5](https://img.shields.io/badge/HTML5-E34F26?logo=html5&logoColor=white) ![CSS3](https://img.shields.io/badge/CSS3-1572B6?logo=css3&logoColor=white) ![JavaScript](https://img.shields.io/badge/JavaScript-F7DF1E?logo=javascript&logoColor=black)

## Features

- **Light/Dark Theme Toggle** - Smooth theme switching with localStorage persistence
- **Scattered Blocks Animation** - Unique building assembly animation in the About section
- **Premium UI Design** - Modern glassmorphism, gradients, and micro-animations
- **Responsive Layout** - Works seamlessly on desktop, tablet, and mobile
- **Project Gallery** - Filterable portfolio with hover effects
- **Custom Cursor** - Interactive cursor with magnetic button effects
- **Scroll Animations** - Reveal animations as you scroll through sections

## Quick Start

1. Clone the repository:
   ```bash
   git clone https://github.com/TuShArBhArDwA/Linark.git
   ```

2. Open `index.html` in your browser

Or use a local server:
```bash
npx serve .
```

## Sections

- **Hero** - Animated intro with stats counter
- **About** - Company info with block assembly animation
- **Services** - Architecture services offered
- **Projects** - Filterable project gallery
- **Team** - Team member profiles
- **Contact** - Contact form and information
